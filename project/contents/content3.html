Githun Desktop을 사용하려면 먼저 프로그램을 다운로드 해야합니다. 

https://desktop.github.com/

 위의 사이트에 들어가서 자신의 운영체제에 맞게 프로그램을 다운로드 하면 됩니다. 그 후 원하는 위치에 설치를 한 다음 실행을 하게 되면 다음과 같은 화면이 나타날 것입니다.

 현재 상태에서 바로 깃허브와 연동하여 사용하려면 먼저 로그인을 합니다. 

 

 

 위와 같은 창에서 로그인을 하면 아래와 같은 화면이 나타나며 상단의 File > New repository 를 클릭하면 새 저장소를 생성할 수 있습니다.

 

 그러면 아래와 같은 창이 나타나는데 여기서 이름과 설명, 컴퓨터에서의 경로, README 파일 생성 여부, .gitignore 파일 설정, 라이센스를 설정할 수 있습니다. 

 

 .gitignore 설정에서 개발하는 언어를 선택할 경우 업로드시 해당 폴더에서 생성되는 개인별 임시폴더, 설정파일 등은 무시하게 됩니다.

 

 

 그러면 해당 경로에 확인해보면 새로운 폴더가 생긴 것을 볼 수 있습니다. 저는 바탕화면으로 설정했기 때문에 폴더가 바탕화면에 생성된 것과 .gitignore 파일이 자동으로 생성된 것을 확인할 수 있습니다.

 

 

 이 상태에서 파일을 추가하면 Github Desktop 을 이용하여 저장소에 업로드 할 수 있습니다. 저는 여기서 hello_world.txt 라는 파일을 만들어서 업로드를 해보겠습니다.

 

 그러면 우측처럼 Github Desktop가 폴더 내의 변경사항이 생긴 것을 자동으로 감지하여 보여줍니다. 이러한 변경 사항을 붉은 원 안의 내용을 작성하여 버튼을 클릭하여 commit이 가능합니다.

 

 

 해당 버튼을 클릭하면, 현재에는 원격 저장소에 repository가 등록이 되지 않은 상태이기 때문에 우측의 Publish reposity를 클릭하여 깃허브에 폴더를 등록하도록 합니다.

 

 

 원격 저장소에 등록을 하고 History 부분을 클릭하면 commit들과 변경부분을 손쉽게 확인 할 수 있습니다.
