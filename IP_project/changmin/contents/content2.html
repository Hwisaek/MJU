Github Desktop은 cmd 환경에서 실행되는 Git 대신 GUI를 이용하여 Github를 이용할 수 있습니다. 

 

위의 그림은 cmd 환경에서 Git을 이용하는 이미지입니다. 해당 이미지를 보면 cmd 환경에서 명령어를 타이핑하여 저장소를 관리하는 것을 볼 수 있습니다.

 

위의 이미지는 Github Desktop을 사용한 GUI 환경에서의 Git입니다. cmd환경과는 다르게 초보자도 쉽게 Github를 이용할 수 있는 환경을 제공해줍니다. cmd 환경에서 사용하는 모든 명령어는 마우스 클릭으로 처리할 수 있습니다.
